Woo Commerce extension

== Description ==

Zero Commerce is an open source extension of the WooCommerce plugin for
Wordpress. Zero Commerce is built and maintained by Zero Daedalus.

Features:
*Added gateways for Google, Amazon, FedEx, USPS, and UPS

== Installation ==

1. Download the Zero Commerce repository into the plugins/ folder of your
Wordpress installation. 

2. Admin Panel -> Plugins -> Installed Plugins -> Activate

3. Enjoy!
